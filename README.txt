Data Analyzer — GUI-анализатор данных на Python

Простое, но мощное приложение для разведочного анализа данных (EDA) с графическим интерфейсом.  
Поддерживает загрузку файлов в форматах `.txt`, `.csv` и `.xlsx`, автоматически адаптируясь к особенностям каждого формата.  
Результаты анализа сохраняются в Excel-файл с несколькими листами, готовый к использованию в Power BI, Excel или дальнейшем анализе.

Возможности

- Загрузка данных из:
  - `.txt` (автоопределение разделителя: `,`, `\t`, `;` и др.)
  - `.csv` (с поддержкой BOM и кириллицы)
  - `.xlsx` (автоматическое определение заголовка)
-  Полный разведочный анализ:
  - Типы данных по колонкам
  - Описательная статистика (`mean`, `std`, `min`, `max`, уникальные значения)
  - Анализ пропущенных значений (количество и %)
  - Обзор категориальных признаков (топ-значения, уникальные категории)
- Визуализация:
  - Гистограммы для всех числовых колонок
- Экспорт отчёта в `*_processed.xlsx` с листами:
  - `Data` — исходные данные
  - `Statistics` — статистика
  - `DataTypes` — типы колонок
  - `MissingValues` — пропуски
  - `CategoricalOverview` — обзор текстовых полей
- Интерфейс на Tkinter:
  - Ввод имени файла
  - Выбор формата
  - Кнопка ручного выбора файла
  - Диалоговые уведомления

Требования

- Python 3.9+
- Windows / macOS / Linux

Установка

git clone https://github.com/Alastrik/Analyzer
cd Analyzer
pip install -r requirements.txt
python Analyzer.py
